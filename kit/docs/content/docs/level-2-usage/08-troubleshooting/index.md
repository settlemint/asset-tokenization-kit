# Troubleshooting & Recovery

**These guides help operators diagnose errors, recover wallets, resolve compliance failures, and escalate incidents with confidence.**

## Roadmap

- [ ] `common-errors.md` — Top 20 error codes with root causes, fixes, and SLA expectations.
- [ ] `transaction-issues.md` — Gas analysis, nonce handling, and transaction replay strategies.
- [ ] `wallet-recovery.md` — Lost access, compromised wallet response, and reassignment workflow.
- [ ] `compliance-failures.md` — Debug blocked transfers and rule conflicts.
- [ ] `performance-issues.md` — Optimise dApp, API, and contract performance.
- [ ] `support-escalation.md` — Escalation matrix, severity levels, and communication templates.

## Delivery Notes

- Each guide must include verification steps so operators know the fix worked.
- Capture log locations, monitoring hooks, and audit evidence for every resolution path.
- Link to relevant operations or integration runbooks when troubleshooting spans multiple domains.
