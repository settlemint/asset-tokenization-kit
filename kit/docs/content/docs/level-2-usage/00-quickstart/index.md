# Quickstart Guides

**Use these quickstarts to move from account access to a live token demonstration in minutes. Each guide follows the Level 2 procedure pattern—objective, prerequisites, numbered steps, verification, and SLA.**

## Roadmap

- [ ] `first-token-10min.md` — ✳ Flagship demo: screenshots for every wizard step, compliance confirmation, and verification checklist for non-technical reviewers.
- [ ] `quickstart-guide.md` — 15-minute platform setup using Bun, Docker Compose, and seeded services with health checks.
- [ ] `video-tutorials.md` — Curation of recorded walkthroughs and future visual assets aligned with the written guides.
