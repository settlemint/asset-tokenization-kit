# Commercials (Deployment & Licensing)

**Clear commercial guardrails let institutions choose delivery models while keeping DALP obligations explicit.** The linked pages summarise what ships in each deployment mode, which licences apply, and where services collateral is still pending approval.

## Navigate the Commercials

- [Deployment Options](./deployment-options) — SaaS, Docker Compose, and Helm pathways with sanctioned migration scripts.
- [Licensing Models](./licensing-models) — Licence matrix covering open-source components, proprietary services, and SaaS subscriptions.
- [Services & Support](./support-tiers) — Professional services and support items awaiting collateral from the Services organisation.

## Implementation Themes

- **Choice without compromise:** Whether SaaS or self-hosted, the same compliance, identity, and lifecycle controls ship, with licensing clarity for each component (Appendix S).
- **Managed migrations:** The SaaS-to-BYOC playbook codifies tenant export, infrastructure staging, dual-write cutover, and post-migration validation so no steps rely on tribal knowledge (Appendix Q).
